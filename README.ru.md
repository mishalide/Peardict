# Project Peardict

Project Peardict — это полностью облачная (и, да, бесплатная) система «предсказаний» для FRC.

Хотите как-то занять себя во время скаутинга? Есть ощущение, что так называемые «рынки предсказаний» — это просто азартные игры под другим названием? В таком случае Project Peardict существует именно для того, чтобы больше не возвращаться к этому вопросу.

**Project Peardict** — это платформа для FRC в формате Kahoot:
- Перед каждым матчем пользователь выбирает альянс или команду, которую считает победителем.
- При верном выборе начисляются очки.
- При неверном выборе не происходит ровным счётом ничего.

---

## Что содержит этот репозиторий

В текущем виде репозиторий включает:
- Бэкенд на Cloudflare Workers (`cloudflare_workers/fujo-worker.js`)
- Интеграционные скрипты Google Apps Script (`appscripts/*.js`)
- Статические HTML-страницы фронтенда (`login.html`, `index.html`)

---

## Краткий технический обзор (действительно краткий)

- Пользователи и матчи загружаются через serverless-функцию (Cloudflare Worker), которая читает данные из Google Sheets и использует слой Google Apps Script для операций записи.
- Фронтенд получает состояние матчей и пользователей через `GET`-запросы и отправляет выборы через `POST`.
- Google Sheets используется как простая и доступная база данных (ровно в том объёме, в котором это имеет смысл).

---

## API, используемые фронтендом

- `GET /api/users`
- `GET /api/matches`
- `GET /api/users/:user_id/bets`
- `POST /api/bets`

*Короткое пояснение для тех, кто ищет скрытый смысл там, где его нет: никаких ставок здесь не совершается. Используется старый API, унаследованный от предыдущего проекта. Азартные игры и «рынки предсказаний» к данному проекту отношения не имеют.*

## Дисклеймер

Project Peardict не связан с реальными ставками или азартными играми. Несмотря на повторное использование бэкенд-API из закрытого экспериментального проекта (Project FUJO), данная платформа является исключительно игровой/квиз-системой:

- Пользователи лишь выбирают предполагаемого победителя.
- Очки начисляются только за правильные ответы.
- Валюты, псевдовалюты и прочих «интересных интерпретаций» здесь нет.
